# Backup of Removed Features & Badges

This document contains a backup of the placeholder sections and badges that were removed from the main `README.md` to clean up the profile. You can copy and paste any of these back into your main profile README if you decide to activate these accounts in the future.

---

## 1. Spotify Now Listening Widget (Static Placeholder)

**Original Code:**
```markdown
## 🎵 Currently Listening To

[![Spotify](https://novatorem.vercel.app/api/spotify?background_color=0d1117&border_color=ffffff)](https://open.spotify.com/user/yourusername)
```
**How to restore:** Replace `yourusername` with your Spotify profile ID (e.g. `31hkxbhhahhvlihccy5hxz4pojvm`) and paste it back into your README.

## 5. Blog Post Feed Placeholders

**Original Code:**
```markdown
## 📚 Latest Blog Posts

<!-- BLOG-POST-LIST:START -->
- [Post Title 1](https://yourblog.com/post-1)
- [Post Title 2](https://dev.to/yourblog/post-2)
- [Post Title 3](https://medium.com/yourblog/post-3)

```
**How to restore:** If you set up an automated blog feed action (like `gautamkrishnar/blog-post-workflow`), place these comments in your README and the action will write your latest posts between the `START` and `END` tags.
